# NeuroSync
a Cyber-Protected Brain-computer solution applied to Robotic systems for Hands-free Biomedical interaction using non-invasive EEG technology and Machine Learning

# Vision 
Help amputees and people with disabiltiies regain mobility of critical functions using non-invasive EMG and EEG technology. This, in a cheap, accessible, and secure way that prevents system override by externals.
